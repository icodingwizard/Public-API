Welcome to simplest Web public API ever !

Overview

API that returns 11 places via places endpoint, which are some of the most beautiful Tourist places to visit in Libya!

Code

Here is simple example using fetch API in JS, feel free to use other libraries like axios or any other language.

            fetch('https://libyantourismapi.vercel.app/places').
then(res => res.json()).
then(data => console.log(data));
